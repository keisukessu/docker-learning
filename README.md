# Docker 学習カリキュラム

## 目標
- 開発環境を Docker で管理できるようになる
- 就職・転職で使えるスキルを身につける
- GitHub に載せられるポートフォリオを作る

## カリキュラム全体像

| モジュール | テーマ | 回数 |
|-----------|--------|------|
| Module 0 | セットアップ確認 | 第1回 |
| Module 1 | Docker とは何か | 第1〜2回 |
| Module 2 | コンテナの基本操作 | 第2〜3回 |
| Module 3 | イメージの理解 | 第3〜4回 |
| Module 4 | Dockerfile でイメージを作る | 第4〜6回 |
| Module 5 | データの永続化 | 第6〜7回 |
| Module 6 | Docker Compose | 第7〜9回 |
| Module 7 | 実践プロジェクト | 第9〜10回 |

## 学習の進め方

各モジュールは **「説明 → デモ → ハンズオン」** の流れで進めます。

- 疑問はその場で聞いてください
- エラーが出ても大丈夫。一緒にトラブルシューティングします
- 実際にコマンドを打って体で覚えましょう
